Configuración JSONBin

Esta versión usa el endpoint estándar de JSONBin para leer el último contenido del bin y sobrescribirlo al subir cambios. Para frontend conviene usar X-Access-Key en vez de X-Master-Key; JSONBin lo recomienda explícitamente porque la Master Key tiene acceso completo y es más apropiada para backend, mientras que las Access Keys se pueden configurar por endpoint y con permisos acotados.
